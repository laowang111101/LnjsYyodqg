# 酒店住房管理系统 java316

## 项目概述

本项目是一个基于Python实现的酒店住房管理系统，采用前后端分离的架构。后端使用Django框架，前端则运用了Vue.js、CSS3、HTML5和JavaScript等技术。

## 技术栈

- 开发语言：Python
- 后端框架：Django
- 前端技术：Vue.js（3.X）、CSS3、HTML5、JavaScript
- 开发工具：PyCharm、Visual Studio Code、HbuildX
- 数据库：MySQL 5.7.26

## 功能模块

系统主要功能模块包括：

- 用户管理
- 房间管理
- 预订管理
- 费用管理
- 报表统计

## 系统角色

- 系统管理员
- 酒店前台
- 客户

## 运行环境

- Python版本：3.0及以上
- 数据库管理工具：phpstudy/Navicat 或 phpstudy/sqlyog

## 部署步骤

1. 环境准备：安装Python环境、MySQL数据库、相关依赖包等
2. 数据库配置：根据系统需求初始化数据库
3. 代码部署：将源码部署到服务器
4. 系统配置：进行必要的系统设置，如数据库连接、静态资源路径等
5. 启动服务：运行Django项目，启动前端服务
